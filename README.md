# Rule
##### AI : 흑, User : 백
##### 승리조건 : 5목 이상 (6목,, 7목 ... 등등)
##### 3 x 3 : 흑,백 모두 금지
##### 4 x 4 : 흑,백 모두 가능
##### 시간제한 : 없음
------------
# Start
##### 시작하면 흑(AI)이 먼저 돌을 놓습니다.
##### 마우스로 백(User)이 놓을 곳을 선택합니다.
##### 승리조건을 만족하면 5초 뒤 재시작합니다.
